📌 Project Overview

**QueueFlow 360** is a full-stack Smart Queue, Appointment & Customer Service Management Platform designed to simplify appointment booking, queue handling, and customer service operations.

The system allows customers to book appointments, generate queue tokens, track live queue status, and view appointment history. Admin users can manage departments, services, customers, and monitor real-time queues.

✨ Features

👨‍💼 Admin Features

- 🔐 Secure Admin Login with JWT Authentication
- 📊 Dashboard with queue statistics
- 🏢 Manage Departments
- 🩺 Manage Services
- 🎫 Monitor Live Queue
- ▶️ Call Next Token
- ✅ Complete Customer Requests
- ⏭ Skip Tokens

👤 Customer Features

- 📝 User Registration & Login
- 📅 Book Appointments
- 🏢 Select Department & Service
- 🎟 Automatic Token Generation
- 📍 Track Live Queue Position
- 📜 View Appointment History
- 🚪 Secure Logout
